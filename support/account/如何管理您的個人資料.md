# 如何管理您的個人資料

「個人資料」頁面 (`/account/settings`) 允許您管理其他使用者在雙龍體育平台上看到您的資訊。

## 1. 存取個人資料設定

1.  登入您的雙龍體育帳號。
2.  點擊右上角的頭像或使用者名稱，然後選擇「**設定**」。
3.  在左側導航欄中，選擇「**個人資料**」。

## 2. 個人資訊設定詳情

您可以隨時更新您的頭像和基本個人資訊。

| 欄位名稱 | 說明 | 操作指南 |
| :--- | :--- | :--- |
| **頭像** | 上傳、更改或移除您的個人頭像。 | 點擊「更換頭像」或「移除」按鈕進行操作。 |
| **顯示名稱** | 您在平台上顯示的名稱。 | 直接在輸入框中修改。 |
| **論壇提及 ID (@ID)** | 唯一的 ID，其他用戶可使用此 ID 在論壇中提及您。 | 僅能包含字母、數字和下劃線。 |
| **自我介紹** | 簡短介紹自己。 | 支援簡單的 **Markdown** 格式。 |
| **國家/地區** | 您的所在地資訊。 | 直接在輸入框中修改。 |
| **網站** | 連結您的個人網站或社群媒體。 | 輸入完整的 URL (例如：`https://example.com`)。 |
| **電子郵件** | 顯示您目前綁定的電子郵件地址。 | 此為您的登入帳號，不可在此頁面直接修改。 |

## 3. 儲存變更

在完成所有修改後，請務必點擊頁面底部的「**存儲資料**」按鈕，以確保您的變更被成功儲存。
