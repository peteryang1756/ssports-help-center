# 雙龍體育帳號管理指南

歡迎使用雙龍體育帳號管理指南。本文件將引導您瞭解如何透過「設定」頁面 (`/account/settings`) 管理您的個人資料、增強帳號安全性以及設定通知偏好。

## 一、個人資料管理

「個人資料」頁面允許您管理其他使用者在雙龍體育平台上看到您的資訊。

### 1. 頭像與個人資訊

您可以隨時更新您的頭像和基本個人資訊。

| 欄位名稱 | 說明 |
| :--- | :--- |
| **頭像** | 上傳、更改或移除您的個人頭像。 |
| **顯示名稱** | 您在平台上顯示的名稱。 |
| **論壇提及 ID (@ID)** | 唯一的 ID，其他用戶可使用此 ID 在論壇中提及您。此 ID 僅能包含字母、數字和下劃線。 |
| **自我介紹** | 簡短介紹自己，支援簡單的 Markdown 格式。 |
| **國家/地區** | 您的所在地資訊。 |
| **網站** | 連結您的個人網站或社群媒體頁面。 |
| **電子郵件** | 顯示您目前綁定的電子郵件地址，此為您的登入帳號。 |

:::tip 提示
請記得在完成任何修改後，點擊頁面底部的「**存儲資料**」按鈕以儲存您的變更。
:::

## 二、帳號安全性設定

「安全性」頁面 (`/account/security`) 提供了多種工具來保護您的帳號安全。

### 1. 變更密碼

為了帳號安全，建議您定期更改密碼。

**密碼要求：**
*   密碼至少需要 **8 個字元**。
*   必須包含**大小寫字母**和**數字**。

**變更步驟：**
1.  在「目前密碼」欄位中輸入您目前的登入密碼。
2.  在「新密碼」欄位中輸入符合要求的新的密碼。
3.  在「確認新密碼」欄位中再次輸入您的新密碼。
4.  點擊「**更新密碼**」按鈕完成變更。

### 2. 雙重驗證 (2FA)

雙重驗證為您的帳號增加了一層額外的安全保護。

*   **功能**：支援使用 **Google Authenticator** 或其他相容的驗證器應用程式進行設定。
*   **設定**：點擊「**設定**」按鈕，按照螢幕上的指示完成 2FA 的啟用流程。

### 3. 登入裝置管理

您可以隨時查看和管理目前登入您帳號的所有裝置。

*   **資訊顯示**：列表會顯示每個登入裝置的詳細資訊，包括裝置類型、瀏覽器、系統、IP 地址、地理位置、首次登入時間、最後活動時間和登入方式。
*   **移除裝置**：
    *   您可以點擊單個裝置旁的「**移除**」按鈕來登出該裝置。
    *   您也可以點擊「**移除其他裝置**」按鈕，一次性登出所有非目前使用中的裝置，以確保帳號安全。

## 三、通知偏好設定

「通知設定」頁面 (`/account/notifications`) 允許您控制接收來自雙龍體育的通知郵件。

### 1. 論壇通知

您可以控制是否接收每週一次的論壇活動摘要郵件。

*   **每週論壇摘要郵件**：
    *   **內容**：彙整您在論壇上的新回覆、提及與讚。
    *   **頻率**：一週寄送一次。
    *   **操作**：您可以透過切換開關來啟用或停用此功能。停用後，您將不會收到每週的摘要郵件。

:::info 備註
此設定旨在幫助您追蹤論壇活動，同時避免訊息被即時通知淹沒。
:::
