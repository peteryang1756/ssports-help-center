# 增強您的帳號安全性

「安全性」頁面 (`/account/security`) 提供了多種工具來保護您的帳號安全，包括密碼變更、雙重驗證和登入裝置管理。

## 1. 變更密碼

為了帳號安全，建議您定期更改密碼。

### 密碼要求

*   密碼至少需要 **8 個字元**。
*   必須包含**大小寫字母**和**數字**。

### 變更步驟

1.  在「目前密碼」欄位中輸入您目前的登入密碼。
2.  在「新密碼」欄位中輸入符合要求的新的密碼。
3.  在「確認新密碼」欄位中再次輸入您的新密碼。
4.  點擊「**更新密碼**」按鈕完成變更。

## 2. 雙重驗證 (2FA)

雙重驗證為您的帳號增加了第二層保護，即使密碼洩露，也能有效防止未經授權的存取。

*   **支援方式**：應用程式驗證器（例如 **Google Authenticator** 或其他相容的驗證器應用程式）。
*   **設定**：點擊「**設定**」按鈕，按照螢幕上的指示掃描 QR Code 並輸入驗證碼以啟用 2FA。

## 3. 登入裝置管理

您可以隨時查看和管理目前登入您帳號的所有裝置，以確保沒有未經授權的設備存取您的帳號。

### 裝置資訊

列表會顯示每個登入裝置的詳細資訊，包括：

*   裝置類型、瀏覽器、系統
*   IP 地址、地理位置
*   首次登入時間、最後活動時間
*   登入方式（例如：電子郵件/密碼）

### 移除裝置

| 操作 | 說明 |
| :--- | :--- |
| **移除** | 點擊單個裝置旁的「**移除**」按鈕，將該裝置登出您的帳號。 |
| **移除其他裝置** | 點擊此按鈕，將**所有非目前使用中**的裝置一次性登出，這是確保帳號安全的快速方法。 |

:::warning 注意
如果您發現任何不認識的登入裝置，請立即將其移除並更改您的密碼。
:::
