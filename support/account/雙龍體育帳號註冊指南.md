# 雙龍體育帳號註冊指南

歡迎加入雙龍體育！本指南將引導您瞭解如何透過 **Signup** 流程建立您的帳號。

## Signup 步驟

請依照以下步驟在雙龍體育平台上建立您的新帳號：

1.  **訪問 Signup 頁面**：
    請在瀏覽器中輸入網址 `https://sysports.de/signup`，或點擊首頁的「註冊」按鈕。

2.  **填寫 Signup 資訊**：
    在註冊表單中，您需要填寫以下四個必填欄位：

| 欄位名稱 | 說明 | 備註 |
| :--- | :--- | :--- |
| **Email** | 您的電子郵件地址。 | 將作為您的登入帳號。 |
| **Password** | 您的登入密碼。 | 建議使用強度高的密碼，包含大小寫字母、數字和符號。 |
| **Username** | 您的使用者名稱。 | 這是您在系統中的唯一識別名稱。 |
| **Display Name** | 您的顯示名稱。 | 這是其他用戶在平台上看到您的名稱。 |

3.  **完成 Signup**：
    確認所有資訊填寫無誤後，點擊「**Register**」按鈕。

4.  **驗證與登入**：
    系統可能會要求您進行電子郵件驗證。完成驗證後，您即可使用您註冊的 Email 和密碼登入雙龍體育平台。

:::tip 提示
如果您在 Signup 過程中遇到任何問題，請隨時聯繫我們的線上客服尋求協助。
:::
