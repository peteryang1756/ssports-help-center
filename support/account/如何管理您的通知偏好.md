# 如何管理您的通知偏好

「通知設定」頁面 (`/account/notifications`) 允許您控制接收來自雙龍體育的通知郵件，幫助您管理資訊流。

## 1. 論壇通知

您可以控制是否接收每週一次的論壇活動摘要郵件。

| 設定項目 | 說明 |
| :--- | :--- |
| **每週論壇摘要郵件** | 彙整您在論壇上的新回覆、提及與讚。 |
| **頻率** | 一週寄送一次。 |
| **操作** | 您可以透過頁面上的**切換開關**來啟用或停用此功能。 |

:::info 備註
啟用此功能可以幫助您追蹤論壇活動，同時避免訊息被即時通知淹沒。停用後，您將不會收到每週的摘要郵件。
:::
