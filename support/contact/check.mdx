# 如何查看和回覆工單消息
:::danger

此文章已過時，尚未更新


:::
在 雙龍體育工單系統中查看和回覆工單消息非常簡單。請按照以下步驟操作：

1. **登入系統**
   訪問 <a href="https://support.ssangyongsports.eu.org">雙龍體育工單系統</a>，點擊“工單”並使用您的 Google 帳號登入或創建帳號。

2. **進入工單頁面**
   在“我的區域”中，可以看到所有工單。

3. **選擇要查看的工單**
   在工單列表中找到您想查看的工單，點擊工單標題進入詳情頁。

4. **查看消息記錄**
   在工單詳情頁面，您可以看到完整的消息記錄，包括客戶和支援人員的所有對話內容。

5. **回覆消息**
   在頁面下方的文字框內輸入您的回覆，然後點擊“發送”按鈕提交。

6. **添加附件（可選）**
   如需上傳文件，請點擊“添加附件”按鈕選擇文件。

7. **設置工單狀態**
   根據需要更新工單狀態，例如“處理中”或“已解決”。

如遇到任何技術問題，請聯繫我們的 IT 支援團隊。
