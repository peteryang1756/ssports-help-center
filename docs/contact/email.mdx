# 聯繫雙龍體育的電子郵件指南
如果您需要聯繫雙龍體育，請按照以下步驟使用電子郵件與我們取得聯繫：

## 1. 打開您的電子郵件客戶端

使用您常用的電子郵件應用程序，例如 Gmail
:::danger

建議不要使用outlook和yahoo與我們聯繫，因為容易被判定為垃圾郵件，建議使用Gmail

:::

## 2. 創建新郵件

點擊「撰寫」或「新郵件」按鈕來創建一封新郵件。

## 3. 填寫郵件信息

- **收件人**：在收件人欄中輸入 `support@ssangyongsports.eu.org`
- **主題**：在主題欄中簡潔明了地描述您的問題或請求。例如：
  - “詢問活動資訊”
  - “技術支援請求”

## 4. 撰寫郵件內容

在郵件正文中，請包括以下信息：

- 您的姓名
- 聯絡電話（可選）
- 詳細描述您的問題或請求

範例郵件內容：

```
親愛的雙龍體育團隊，

我想詢問有關即將舉行的活動的資訊。請提供詳細的時間和地點。

謝謝！

[您的姓名]
[您的聯絡電話]
```

## 5. 檢查郵件

在發送之前，請檢查郵件內容是否正確，確保沒有拼寫錯誤。

## 6. 發送郵件

點擊「發送」按鈕，您的郵件將寄送至雙龍體育的客服團隊。

## 7. 等待回覆

我們會在收到您的郵件後盡快進行回覆，<a href="https://ssangyongsports.eu.org/docs/contact/speed">詳細說明</a>

如有任何其他問題，隨時聯繫我們！
